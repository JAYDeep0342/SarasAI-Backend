## ✨ Features

- 🧠 **Gemini API Integration** - Leverage Google's advanced AI models
- 🔐 **Spring Security** - JWT-based authentication
- 💾 **PostgreSQL** - Robust relational database
- 🏗️ **Dependency Injection** - Clean Spring IoC architecture
- 📊 **JPA/Hibernate** - Simplified database operations
- 🔄 **RESTful APIs** - Well-structured REST endpoints
- 🛡️ **Error Handling** - Comprehensive exception management

## 📋 Prerequisites

Before you begin, ensure you have:

- **Java 17+** installed ([Download](https://www.oracle.com/java/technologies/downloads/))
- **Maven 3.6+** installed ([Download](https://maven.apache.org/download.cgi))
- **PostgreSQL 12+** running locally or remotely ([Download](https://www.postgresql.org/download/))
- **Google Gemini API Key** ([Get Key](https://makersuite.google.com/app/apikey))
- **Git** for version control
